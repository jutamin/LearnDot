## 🛠️ Tech Stack

| 영역 | 기술 | 설명 |
|------|------|------|
| UI 프레임워크 | SwiftUI | 선언형 UI로 접근성과 유지보수성 향상 |
| 아키텍처 | MVVM | UI와 비즈니스 로직 분리로 구조적 설계 |
| 접근성 | VoiceOver | 시각장애인을 위한 음성 안내 및 인터랙션 지원 |
| 점자 처리 | KorToBraille | 한글을 점자로 변환하는 라이브러리 |
| 데이터 관리 | BrailleData | 점자 규칙 및 패턴을 관리하는 커스텀 데이터 |
| 버전 관리 | Git, GitHub | 브랜치 전략 기반 협업 및 코드 관리 |
| 협업 도구 | Figma, Notion | 디자인 및 문서 기반 협업 |
| 기준 문서 | 2024 한국 점자 규정 | 국립국어원 개정 점자 규정 기반 학습 |
